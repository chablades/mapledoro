# MapleDoro

Just another MapleStory website for seeing events, sharing characters and more!

## Local setup

1. Install Node.js 20+ (LTS recommended).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:3000`.

## Build and run

```bash
npm run build
npm run start
```
